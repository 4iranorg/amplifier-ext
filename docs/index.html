How It Works

1

Install

Add the extension from Firefox Add-ons and enter your API key

2

Browse

Find posts about Iran on X (Twitter)

3

Amplify

Click the Amplify button to generate response options

4

Review & Post

Choose a response, refine if needed, and post

You Control the Message

Iran Amplifier is a tool that assists you — it never posts anything automatically. This extension does not automate posting, coordinate users, or generate identical messages.

Talking Points Choose which facts the AI can reference
Calls to Action Select which policy demands to advocate for
Style Customize the tone and approach
AI Service OpenAI or Anthropic — your choice
Review Every response shown to you before use
Post Only you decide what gets published

Is It Free?

The extension is free and open source. AI services (OpenAI or Anthropic) charge small fees per response — typically 1–5 cents each.
